ğŸ“Œ Project Overview
This project simulates a real-world TikTok analytics platform using a synthetic dataset of 19,382 videos. The goal is to build predictive models that help content creators and platform managers make data-driven decisions.

Key objectives:

Predict engagement rate (likes + shares + comments / views) with high accuracy.

Classify videos into risk levels (low, medium, high) based on content and author history.

Identify optimal content strategies (duration, posting time, topics) for maximum reach.

Forecast daily view trends to aid in capacity planning.

All models are trained, evaluated, and saved for deployment. A Streamlit dashboard and REST API are provided for real-time predictions.

Note: This project is for educational and portfolio purposes. The dataset is synthetically generated to mimic real TikTok patterns.
